# Veterinary-Clinic-Performance-Patient-Analysis-Dashboard
A comprehensive Power BI dashboard built to analyze the operational and business performance of a veterinary clinic. This project demonstrates data cleaning, data modeling, and interactive dashboard design using a real-world healthcare business scenario.

📌 Project Overview
Veterinary clinics generate large amounts of data related to appointments, doctors, pets, customers, treatments, and revenue. This dashboard transforms raw clinic data into meaningful business insights that help management monitor performance and make data-driven decisions.
The project focuses on:
Clinic performance analysis
Revenue tracking
Appointment trends
Doctor performance
Customer insights
Pet demographics

🎯 Business Objectives
Monitor overall clinic performance.
Track appointment volume over time.
Analyze revenue generated from treatments.
Evaluate doctor performance.
Understand customer and pet demographics.
Identify business trends to support better decision-making.

🛠️ Tools & Technologies
Power BI Desktop

Data Modeling (Star Schema)
